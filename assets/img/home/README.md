# 首页素材（与 Figma「首页」Frame 对应）

将设计稿导出的切图放在本目录，页面会按文件名引用：

| 文件（可选） | 用途 |
|-------------|------|
| `logo.svg` / `logo.png` | 左上角品牌（默认已提供 `logo.svg` 占位） |
| `banner.jpg` / `banner.png` | 右侧 Banner 大图（未提供时使用 `banner.svg` 渐变占位） |
| 其它 | 可按需在 `pages/index.vue` 或 `layouts/home.vue` 中增加引用 |

替换同名文件即可覆盖占位图。
